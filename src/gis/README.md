# GIS 기반 외국인 밀집지역 분석

서울시 외국인 생활인구 + S-DoT 유동인구 데이터를 활용한 **Hub & Spoke 물류 전략** 수립 모듈

## 기술 스택

| 분류 | 기술 |
|------|------|
| 데이터 처리 | Pandas, NumPy |
| 데이터 소스 | 서울 열린데이터광장 (생활인구), S-DoT (유동인구 센서) |
| 분석 방법 | 시간대별 집계, 복합점수, MinMax 정규화 |

## 분석 목적

> 다이소 영업시간(10~22시) 기준으로 외국인이 가장 많이 체류하는 자치구를 식별하고,
> **Hub(관광 상권)** vs **Spoke(거주 상권)** 재고 배치 전략을 수립한다.

## 3가지 분석 방법론

```
┌──────────────────────────────────────────────────────────────────┐
│                    외국인 생활인구 데이터                          │
│              (2024.01 ~ 2025.09, 월별 CSV)                       │
│              영업시간 필터: 10시 ~ 22시 (13시간)                   │
└──────────────────────────────┬───────────────────────────────────┘
                               │
              ┌────────────────┼────────────────┐
              ▼                ▼                ▼
┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐
│  방법 A           │ │  방법 B           │ │  방법 C           │
│  시간대별 분석    │ │  평균 스냅샷      │ │  Person-Hour     │
│                  │ │                  │ │                  │
│  구별 × 시간대   │ │  Σ(10~22시) /    │ │  Σ(10~22시) /    │
│  일평균 집계     │ │  13시간 / 일수    │ │  일수            │
│  피크타임 감지   │ │                  │ │                  │
│                  │ │  → 구별 순위 비교 │ │  → 체류 가치 분석│
│  → 시간대 전략   │ │  → 가장 직관적   │ │  → B × 13 ≈ C   │
└──────────────────┘ └──────────────────┘ └──────────────────┘
                               │
                               ▼
                    ┌──────────────────────┐
                    │  S-DoT 유동인구 결합  │
                    │  (101개 센서, 주간)   │
                    └──────────┬───────────┘
                               │
                               ▼
                    ┌──────────────────────┐
                    │  복합점수 계산        │
                    │  MinMax(외국인)       │
                    │  + MinMax(유동량)     │
                    │  = 0 ~ 2점           │
                    └──────────┬───────────┘
                               │
                    ┌──────────┴───────────┐
                    ▼                      ▼
            ┌──────────────┐      ┌──────────────┐
            │   Hub (30%)  │      │  Spoke (70%) │
            │  관광 상권    │      │  거주 상권    │
            │              │      │              │
            │  스킨케어    │      │  구색(Variety)│
            │  5~10배 재고 │      │  위주 진열    │
            └──────────────┘      └──────────────┘
```

## 핵심 계산식

| 지표 | 공식 | 용도 |
|------|------|------|
| 외국인 일평균 (방법 B) | Σ(10~22시 외국인) / 13시간 / 일수 | 구별 밀집도 순위 |
| Person-Hour (방법 C) | Σ(10~22시 외국인) / 일수 | 체류 가치 분석 |
| S-DoT 일평균 | Σ 방문자수 / 일수 | 유동인구 규모 |
| 복합점수 | MinMax(외국인) + MinMax(유동량) | Hub/Spoke 분류 |
| Hub 임계값 | 복합점수 상위 30% (70분위수) | 전략 분류 기준 |
| 검증 | 방법 B × 13 ≈ 방법 C | 수학적 일관성 확인 |

## 분석 결과: 구별 우선순위

| 순위 | 자치구 | 특징 | 전략 |
|------|--------|------|------|
| 1 | **중구** | 명동, 관광 중심, 외국인 1위 | Hub: 다국어 서비스, 스킨케어 집중 |
| 2 | **강남구** | 쇼핑, 비즈니스 | Hub: 프리미엄 라인업 |
| 3 | **용산구** | 이태원, 다문화 | Hub: 다양한 언어 대응 |
| 4 | **구로/금천** | 중국인 밀집 | Spoke: 중국어 전용 매대 |
